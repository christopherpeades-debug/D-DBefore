# D&D Before

Interactive D&D 3.5 character sheet.

## Install

Download the latest `D&D_Before_v1.21_Setup.exe` from [Releases](https://github.com/christopherpeades-debug/D-DBefore/releases) and run the installer.

- Default install location: `C:\D&D Before`
- Character data is saved next to the install folder (not AppData)
- Installer uses a dark-mode wizard
- Upgrading removes legacy versioned install folders (v1.1, v1.11, v1.12, and others) automatically

Current version: **v1.21**

> Use the in-app **Hamburger menu → Check for Updates** after install. Updates compare release tags on GitHub (e.g. `v1.21`).
