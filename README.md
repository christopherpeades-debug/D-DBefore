# D&D Before

Interactive D&D 3.5 character sheet.

## Install

Download the latest `D&D_Before_v1.12_Setup.exe` from [Releases](https://github.com/christopherpeades-debug/D-DBefore/releases) or from this repository and run the installer.

- Default install location: `C:\D&D Before v1.12`
- Character data is saved next to the install folder (not AppData)
- Installer uses a dark-mode wizard
- Upgrading from v1.1, v1.11, or v1.2 removes those old install folders automatically

Current version: **v1.12**

> **Note:** v1.12 replaces the v1.2 release. Version numbers are compared numerically, so `1.2` was treated as older than `1.11` and the in-app updater would not offer it. Use v1.12 for updates from v1.11.
