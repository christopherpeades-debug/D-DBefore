# D&D Before

Interactive D&D 3.5 character sheet.

## Install

Download the latest `D&D_Before_v1.2_Setup.exe` from [Releases](https://github.com/christopherpeades-debug/D-DBefore/releases) or from this repository and run the installer.

- Default install location: `C:\D&D Before v1.2`
- Character data is saved next to the install folder (not AppData)
- Installer uses a dark-mode wizard
- Upgrading from v1.1 or v1.11 removes those old install folders automatically

Current version: **v1.2**
